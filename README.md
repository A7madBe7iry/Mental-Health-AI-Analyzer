🧠 AI Mental Health Analyzer

📌 Overview
An AI-powered mental health analysis system that uses Natural Language Processing (NLP) and Machine Learning to analyze user input and detect potential signs of stress, anxiety, and depression. The system provides intelligent insights to support early mental health awareness.

🚀 Features
Text-based mental health analysis using NLP
Machine Learning-based classification
Detection of stress, anxiety, and depression patterns
Simple and interactive Streamlit interface
Real-time predictions

🛠️ Tech Stack
Python
Machine Learning
NLP
Streamlit
Scikit-learn
Pandas / NumPy

📊 How It Works
User enters text input
Text is preprocessed and cleaned
ML model analyzes patterns
System outputs predicted mental state with insights
